– Alle Versionen sind ‚entgoogled‘. Das heißt, das OS sendet keine unerwünschten Daten an Google Server

- All versions are 'de-googled'. This means that the OS itself does not send unwanted data to Google Server

– Alle LeOS versionen sind Vanilla Versionen, also ohne jegliche Google Play Services.

- All LeOS versions are vanilla versions, so without any Google Play Services.

– Gapps funktionieren NICHT in LeOS

- Gapps do NOT work in LeOS

– Als Google Play Services Ersatz ist eine LeOS-MicroG Version integriert. Damit

- keine Geräte Registrierung,

- kein anlegen eines Google Accounts,

- keine Push Nachrichten über Google Push server

- A LeOS MicroG version is integrated as Google Play Services replacement. With it

- no device registration,

- no creation of a Google account,

- no push messages via Google Push server
